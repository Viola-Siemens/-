340  public static DragAndDropVisualMode HandleClipPaneObjectDragAndDrop(IEnumerable<UnityObject> objectsBeingDropped, TrackAsset targetTrack, bool perform, TimelineAsset timeline, TrackAsset parent, PlayableDirector director, double candidateTime, TypeResolver typeResolver, TrackAsset insertBefore = null)
341  {
342  if (timeline == null)
343  return DragAndDropVisualMode.Rejected;
344 
345  // locked tracks always reject
346  if (targetTrack != null && targetTrack.lockedInHierarchy)
347  return DragAndDropVisualMode.Rejected;
348 
349  // treat group tracks as having no track
350  if (targetTrack is GroupTrack)
351  {
352  parent = targetTrack;
353  targetTrack = null;
354  }
355 
356  // Special case for monoscripts, since they describe the type
357  if (objectsBeingDropped.Any(o => o is MonoScript))
358  return HandleClipPaneMonoScriptDragAndDrop(objectsBeingDropped.OfType<MonoScript>(), targetTrack, perform, timeline, parent, director, candidateTime);
359 
360  // no unity objects, or explicit exceptions
361  if (!objectsBeingDropped.Any() || objectsBeingDropped.Any(o => !ValidateObjectDrop(o)))
362  return DragAndDropVisualMode.Rejected;
363 
364  // reject scene references if we have no context
365  if (director == null && objectsBeingDropped.Any(o => o.IsSceneObject()))
366  return DragAndDropVisualMode.Rejected;
367 
368  var validTrackTypes = objectsBeingDropped.SelectMany(o => TypeUtility.GetTrackTypesForObject(o)).Distinct().ToList();
369  // special case for playable assets
370  if (objectsBeingDropped.Any(o => TypeUtility.IsConcretePlayableAsset(o.GetType())))
371  {
372  var playableAssets = objectsBeingDropped.OfType<IPlayableAsset>().Where(o => TypeUtility.IsConcretePlayableAsset(o.GetType()));
373  return HandleClipPanePlayableAssetDragAndDrop(playableAssets, targetTrack, perform, timeline, parent, director, candidateTime, typeResolver);
374  }
375 
376  var markerTypes = objectsBeingDropped.SelectMany(o => TypeUtility.MarkerTypesWithFieldForObject(o)).Distinct();
377 
378  // Markers support all tracks
379  if (!markerTypes.Any())
380  {
381  // No tracks support this object
382  if (!validTrackTypes.Any())
383  return DragAndDropVisualMode.Rejected;
384 
385  // no tracks for this object
386  if (targetTrack != null && !validTrackTypes.Contains(targetTrack.GetType()))
387  return DragAndDropVisualMode.Rejected;
388  }
389 
390  // there is no target track, dropping to empty space, or onto a group
391  if (perform)
392  {
393  // choose track and then clip
394  if (targetTrack == null)
395  {
396  var createdTrack = HandleTrackAndItemCreation(objectsBeingDropped, candidateTime, typeResolver, timeline, parent, validTrackTypes, insertBefore);
397  if (!createdTrack)
398  {
399  timeline.CreateMarkerTrack();
400  HandleItemCreation(objectsBeingDropped, timeline.markerTrack, candidateTime, typeResolver, true); // menu is always popped if ambiguous choice
401  }
402  }
403  // just choose clip/marker
404  else
405  {
406  HandleItemCreation(objectsBeingDropped, targetTrack, candidateTime, typeResolver, true); // menu is always popped if ambiguous choice
407  }
408  }
409 
410  return DragAndDropVisualMode.Copy;
411  }

610  public static void HandleBindingDragAndDrop(TrackAsset dropTarget, Type requiredBindingType)
611  {
612  var objectBeingDragged = DragAndDrop.objectReferences[0];
613 
614  var action = BindingUtility.GetBindingAction(requiredBindingType, objectBeingDragged);
615  DragAndDrop.visualMode = action == BindingAction.DoNotBind
616  ? DragAndDropVisualMode.Rejected
617  : DragAndDropVisualMode.Link;
618 
619  if (action == BindingAction.DoNotBind || Event.current.type != EventType.DragPerform)
620  return;
621 
622  var director = TimelineEditor.inspectedDirector;
623 
624  switch (action)
625  {
626  case BindingAction.BindDirectly:
627  {
628  BindingUtility.Bind(director, dropTarget, objectBeingDragged);
629  break;
630  }
631  case BindingAction.BindToExistingComponent:
632  {
633  var gameObjectBeingDragged = objectBeingDragged as GameObject;
634  Debug.Assert(gameObjectBeingDragged != null, "The object being dragged was detected as being a GameObject");
635 
636  BindingUtility.Bind(director, dropTarget, gameObjectBeingDragged.GetComponent(requiredBindingType));
637  break;
638  }
639  case BindingAction.BindToMissingComponent:
640  {
641  var gameObjectBeingDragged = objectBeingDragged as GameObject;
642  Debug.Assert(gameObjectBeingDragged != null, "The object being dragged was detected as being a GameObject");
643 
644  var typeNameOfComponent = requiredBindingType.ToString().Split(".".ToCharArray()).Last();
645  var bindMenu = new GenericMenu();
